BBC’n gareewwan WhatsApp kanneen keessaa tokko keessatti miseensota ‘raayyaa miidiyaa’ fi aangawoota gidduu ergaawwaniifi ajajawwan darbaa turan akkasumas duulawwan gaggeeffaman hordofaa tureera.

Kanneen garee kana hoogganan [admin] suuraawwan kana gaafa ergu miseensonni garee kanaa suuraawwan kana fuulawwan fesbuukii ''informal’’ ta’aniirratti ''bal’inaan’’ akka facaasan ajaja dabarseera.
Ergaan miseensota ‘raayyaa miidiyaaf’ ergame ''ergaa kana fuulawwan [fesbuukii] informal ta’an hundarratti bal’inaan akka tamsa’u dammaqinaan qindeessaa. Caasaan keenya qooda hin fudhanne akka hin jiraanne. Hunduu haa hirmaatu!'' ajaja jedhu kan qabatedha.

Suuraawwan garee WhatsApp irratti ergaman kanneen fesbuukiirratti qoodamaa turan waliin wal fakkaatoodha. Barreeffamoonni sadan garuu miseensota garee kanaaf akka hin ergamne BBC’n hubateera.
Raayyaan miidiyaa paartii Badhaadhinaa duula kana keessatti akka hirmaatan kan taasise caasaa aanaa paartii kutaa magaalaa Qirqoos qofa miti.

''[Barreeffamni gareerratti ergame] lammummaa Abuuna Pheexiroos ibsuun ‘biyyaa baqatan’ jedha. ‘Lammummaa alaa qaban dahoo godhachuun sanadoota sobaa garagaraa qopheessuun biyyaa karaa seeraan alaa bahaniiru’ jedha,’’ jechuun barreeffamni garee WhatsApp irratti gadhiifame fesbuukiirratti kan faca’e waliin akka wal fakkaatu BBC’f mirkaneessaniiru.
Dhimmi Abuuna Pheexiroos duula fesbuukii tokkicha qofa qindeessummaa caasaa aanaa waajjira paartii Badhaadhinaa magaalaa Finfinneen taasifame miti.
Caasaaleen aanaa waajjira paartii Badhaadhinaa magaalaa Finfinnee maqaa mootummaa bareechuuf barreeffamoota, suuraafi viidiyoo bifa duulaan irra deddeebiin faca’aa akka ture BBC’n mirkanesseera.
BBC’n gareewwan telegiraamii shan biroo caasaaleen aanaa paartii Badhaadhinaa kutaalee magaalaa Aqaaqii Qaallittii, Nifaas Silk Laaftoo, Lammii Kuraa, Gullalleefi Kolfee Qaraaniyoo ergaa itti wal ergan argateera.
Miseensonni Paartii Badhaadhinaa BBC’n dubbise akkasumas hoggantonniifi ogeeyyiin kominikeeshinii dhaabbilee mootummaa federaalaa fi Finfinnee keessa dalagan duulawwan fesbuukii kan gaggeeffaman miseensota ‘raayyaa miidiyaa’ paartiichaa fi hoggantoota aanaaleen akka ta’e ibsaniiru.
Duulawwan kan taasifamanis ''seenessi’’ mootummaafi paartii ''olaantummaa akka qabaatan’’ akka ta’e maddeen dubbataniiru.
‘Raayyaa miidiyaa’ jedhamuun kan waamaman liigii dargaggootaafi dubartootaa keessatti miseensa ta’un kan hirmaatan akkasumas namoota ‘hawaasa dammaqsu’ jedhaman akka ta’an BBC’n namoota dubbiseerraafi ergaawwan garee WhatsApp fi telegiraamii gubbaa walii qoodaman irraa hubateera.
Hoggantoonni aanaalee magaalaa Finfinnee sochii akkanaarratti akka hirmaatan nami hoggansa aanaa keessa jiran akkasumas maddeen hoogganaafi ogeessa kominikeeshinii ta’an afur BBC’tti dubbataniiru.

Ergaa gareewwan WhatsApp fi telegiraamii waajjira paartii aanaa kutaalee magaalaa Qirqoos fi Yakkaa keessatti walii ergaman irratti hordoffii BBC’n taasise waan maddeen jedhan kan mirkaneessudha.
Garee telegiraamii aanaa kutaa magaalaa Yakkaa jala argamuufi miseensota 30 ol qabu keessatti ergaawwan qoodaman, miseensonni garichaa waajjiraalee bulchiinsa aanicha garagaraa keessatti hooggansarra akka jiran kan agarsiisudha. Namoonni BBC’n hooggansa aanaa keessa akka jiran mirkaneesse garicha keessa jiru.
Hoggantoonni aanaalee magaalattii keessatti argaman hundi ''kallattii’’ paartiirraa kennameen akkaawuntii fesbuukii tokko caalaa akka qabaatan taasifamuu maddeen sadii BBC ibsaniiru.
Akkaawuntiin dabalataa akkanaa gareewwan WhatsApp fi telegiraamii irratti ''fuula ykn peejii informaal’’ jedhamee eerama.
Hirmaannaan hooggantoonni aanaa miidiyaa hawaasummaarratti taasisan fedhiin qofa kan taasifamu akka hin taane odeeffannoowwan BBC’n argate ni mul’isu.

Gamaaggamni hojii aanaalee Finfinnee keessatti taasifamuufi ''kallattiin siyaasaa’’ kennamu ''koree qindeessituu hooggansa koorii [core]’’, namoota shan caasaa bulchiinsa aanaa akkasumas paartiirraa walitti dhufan of keessatti qabateen akka ta’e hooggansi aanaa tokko ibsaniiru.
Gamaggama koree kanaan taasifamuurratti itti-fayyadamni miidiyaa hawaasummaa ajandaa tokko ta’e akka dhihaatu kan dubbatan maddi kun, ''kaampeeyinii [duula] godhaa jedhamee ajandaa kennameerratti [miseensi hooggansa tokko] yoo hin hirmaanne itti gaafatama.
''Hooggansa [aanaa] 30 wayii keessaa hammana kan ta’u hirmaateera’ jedhama. Iskiriin shoot [suuraa kaafamee] gara gubbaatti ergama. Gara kutaa magaalaa [dabarfamee] abaluun hirmaateera, abaluun hin hirmaanne’’ jedhama jechuun dhimma itti gaafatamummaa hordofsiisu akka ta’e hubachiisu.
Kutaa magaalaattis ta’e gamaggamni kun akka adeemsifamu hoogganaan kominikeeshinii kutaa magaalaa tokko BBC’tti dubbataniiru.
Hoggantoonni paartii gamaaggama gaggeessan, maxxansoota fuulawwan mootummaa fi paartiirra jiran ilaalchiee ''maaliif like [jaalachuu], share [qooduu] hin gootan,’’ gaaffii jedhu hirmaattotaaf akka dhiheessan ibsaniiru.

Gareewwan WhatsApp fi telegiraamii caasaalee siyaasaa aanaalee garagaraa irrattis hirmaannaan miidiyaa hawaasummaa miseensotaa irra deddeebiin ni ka’a.
Miseensonni raayyaa miidiyaa iskiriin shootii ‘like, comment fi share’ gochuu isaanii agarsiisu akka ergan akka gaafataman ergaawwan garee kana keessatti walii deebiifaman irraa BBC’n hubateera.
Miseensonni hooggansa aanaaleefi raayyaa miidiyaa linkii akka ergan akkasumas miseensonni paartii akkaawuntii fesbuukiin isaanii cufame [lock] barbaachisummaa ‘inspection [hordoffiif]’ jecha akka banan ergaawwan gareewwan kana keessatti dabarfamanii mul’ataniiru.
Miseensonni hooggansa aanaalee bulchiinsa magaalaa Finfinnee sochii miidiyaa hawaasummaarratti taasisaniifi tatamsa’ina oduu sobaafi haasaa jibbinsaa keessatti qooda qaban ilaalchisee yaada bulchiinsa magaalichaa argachuuf yaaliin BBC irra deddeebiin taasise hin milkoofne.
Hooggantuun Biiroo Kominikeeshinii Magaalaa Finfinnee Aadde Innaataalam Mallasa, gaaffiin xalayaadhaan akka isaaniif dhihaatu gaafatanii BBC’n xalayaan dhiheessuus isaan garuu deebii hin kennine.
Waajjirri muummee Paartii Badhaadhinaa gama isaan deebii barreeffamaan BBC’f kenneen caasaa paartichaa keessatti gurmaa’inni ‘raayyaa miidiyaa’ jedhamu akka hin jirre ibsuun dhimmicha waakkateera.
Paartichi maqaafi fudhatamummaa mootummaa akkasumas aangawoota olaanoo karaa nam-tolcheen guddisuufis ''kaayyoo miidiyaa hawaasummaa idileerraa adda ta’een sochii adda akka taasisu’’ deebii kenneen ibseera.
Miseensonni paartiichaa ''fuulawwanis ta’e akkaawuntii dhuunfaasaan irratti ejjennoowwan paartichaa akka tarkaanfachiisan’’ fi paarticharratti ''wallaalummaanis ta’e beekaa duula banaman qajeelfamoota paartichaarratti hundaa’un deebii akka kennan akka jajjabeessu,’’ ibseera.
Waajjirri muummee kun ''miidiyaa hawaasummaa gaarummaan fayyadamuun ala [miseensonni paartichaa] akkaawuntii sobaa akka bananis ta’e ergaawwan tokko [monotonous] akka dabarsan tasa hin godhu,’’ jechuun ofirraa ittise.
Jijjirama siyaasaa waggaa jahaan dura bulchiinsi Ministirri Muummee Abiy Ahimad gara aangootti akka dhufu taasisaniin booda, dhiibbaan miidiyaa hawaasaa siyaasa Itoophiyaa keessatti qabu ‘‘baayyee olaanaa’’ ta’uu, Daareektarri Olaanaan Wiirtuu Mirgootaafi Guddina Dimokiraasi, Obbo Bafiqaadu Haayilu ni dubbatu.
Mormiin keessumaa bara 2008’tti Naannoo Oromiyaatti cimee ture keessumaa kan inni hoogganame rogeeyyii siyaasaa fesbuukii irratti tooftaa fi kallattii mormii agarsiisaa turaniin.
Mormii kanaan gara aangootti kan dhufe bulchiinsi MM Abiy Ahimad, yeroo ammaatti ‘‘rakkoon fudhatamummaa kan mudateen akka fakkaatu’’ Obbo Bafiqaaduun ni dubbatu.

Obbo Bafiqaaduun akka jedhanitti, ‘‘baay’ina ummatasaaniin guddaa kan ta’an akka naannoo Oromiyaa, Amaaraa akkasumas naannoo Somaalee dhiibbaan isaa baayyee olaanaa ta’e keessatti tasgabbii dhabiinsi mul’atu’’ fudhatamni haphachuu isaaf sababawwan ijoodha jedhu.
‘‘Nageenyi bora’u kana waliin wal qabatee rakkoolee diinagdee, sarbamoota mirga namoomaa, daangeeffamu sochii lammiilee fi rakkoolee biraa baayyeen fudhatama mootummaa miidhaa dhufeera.
‘‘Olaantummaan seeraa laaffateera, [akkasumas] malaammaltummaan dabalaa jira jedhan baayyee jiru,’’ jechuun sababoota biraas tarreessu.
Sababoota kanneeniin komiiwwan ummata biratti uumaman miidiyaalee hawaasummaa irratti calaqqisiifamuun isaanii ‘‘ilaalcha (impression) MM deeggartootaafi jaallattoota baayyee hin qaban jedhu’’ uumeera jedhanii akka amanan ibsu Obbo Bafiqaaduun.
Maxxansawwan Ministira Muummee jala ykn viidiyoowwan isaan irratti mul’atan jala ‘‘yaadawwan leellisuun itti baayyatu, dhugaafi kan uumamaa hin fakkaanne’’ yoo kennaman hubachu kan dubbatan Obbo Befiqaaduun, sochiin kunis ‘‘mala fudhatamaa fi kabaja’’ mootummaan sababii adda addaatiif dhabe ‘‘oomishu’’ akka fakkaatu himu.

ALI Amajjii 8 fi 9, 201 sagantaa kutaa walitti fufiinsa qabu lamaan dhiyaate kanaanis, kan wayita Ministirri Muummee Abiy Ahimad ‘‘pirojektoonni gurguddaan’’ magaalaa Finfinnee keessatti ijaaramaniif adeemsa irratti argaman aangawoottan isaanii daawwachiisan ittiin agarsiifameedha.
Guyyoota sagantaaleen kunneen itti tamsaafamaniitti, miseensonni garee WhatsApp barreeffamoota Afaan Amaaraa, Oromoofi Ingiliziitiin qophaa’an 128 toora EBC irratti akka yaada kennaniif ergameefiira.
BBCn xinxala yaadawwan sagantaa guyyaa jalqabaa darbe irratti kennameerratti taasiseen, barreeffamoonni miseensota garee WhatsApp’f ergaman 9 yeroo 108 yaada kennamuun argamaniiru.
Yaadawwan kutaa tokkoffaa sagantichaa irratti kennaman, barreeffamoonni caasaalee Paartii Badhaadhinaatiin kan qophaa’an fakkaatanis yeroo baayyee yaada akka kennan (Comment) akka godhan kan agarsiisudha.
Sagantaan kun wayita tamsa’utti akkaawuntiiwwan facebook fi marsariitiiwwan barreeffama walfakkaataa irra deddeebiin yoo yaada kennan mul’ata.

Barreeffamoota kanneen keessaayis tokko ''qabeenya uumamaa kan hubatan Ministirri Mummee Dr Abiy Ahimad eddoowwan uumamaa bareechuu fi bu’uraalee misoomaa guutuufiin akkasumas bakka boqonnaa turistootaa ammayyaa hojjachuurratti argamu’’ kan jedhudha.
Akkaawuntiiwwan Fesbuukii garaagaraas barreeffama kana yoo xinnaate yeroo 78 komenti taasisaniiru. Barreeffamoonni Afaan Ingiliiziin qophaa’an gara biraatis yeroo 100 ol akka yaadaatti kennamaniiru.
Waajjirri Paartii Badhaadhina Magaalaa Finfinnee, keessattuu barreeffamoonni, Suuraalee fi viidiyoowwan pirojektoota bulchiinsi magaalaa hojjachiisu dinqisiifatan qopheessuudhaan, haala duulaatiin irra dedeebiin akka tamsaasu BBC hordoffii garee WhatsApp Badhaadhina aanaa irratti taasiseen mirkaneesseera.

Sirna eebba Godaambaa Yaaddannoo Adawaa baatii Guraandhalaa keessa raawwate torbaniin dura, duulawwan pirojakticha leellisan gaggeeffamaniiru.
Bitootessa keessattimmoo sababii misooma koridarii daandii bulchiinsi magaalaa gaggeessaa jiruutiin, namoota gamoowwaniifi manneen naannawa Piyaasaatii diigaman balaaleeffatan kan qeeqan, suuraalee bifa duulaatiin facebook gubbaatti akka qoodaman taasifamaniiru.
Yaadawwan miidiyaa hawaasummaa irratti ‘‘namoota baayyee hiriirsuudhaan daboodhaan (mob)’’ qoodaman to’achuudhaaf haala qindaa’een dulliiwwan taasifaman, gara ‘‘yaadaa ykn dogongorsuutti’’ akka jijjiraman Awustiraaliyaa, Yunivarsitii Moonaashiitti qorataa Seera Mirgoota Namoomaa Dijitaalii, Dr Yoohaannis Iniyew ni dubbatu.
Dr Yohaannis akka jedhaniitti, akkaawuntiiwwan mootummaa deeggaran sochiileen ‘’lammii sirrii fakkaachuun’’ taasisan keessattuu tibba filannootti yaada namoota filatanii burjaajjeessuu akkasumas ‘‘bu’aa dandeettii anamtaa fi murteessuu miidhuu’’ danda’a.
Kana malees sochiin miidyaa hawaasummaa ''bal’inaan to’achuun’’ taasifamu, mala ''namoonni osoo mootummaa hin deeggarsa yaadasaanii si’ominaan barreessan waltajicha keessaa akka bahan’’ ittiin taasifamu ta’u dubbatu qorataan kun.
Duula ''haasaa jibbinsaa'' fakkaatan mormitootaafi qeeqxota irratti taasifaman
Hojiin ''raayyaa miidiyaa hawaasaa’’ maqaa mootummaa ykn paartii leellisuu qofa miti. Fuula Feesbuukii yaada mootummaa mormu qoodan irratti miseensonni ''raayyaa miidiyaa hawaasaa’’ barreeffama qopha’ee isaaniif kenname yaada akka kennanii (komeent) akka godhan BBC’n mirkaneesseera.
Haala kanaan yaada Fesbuukii irratti qoodaniin barreefamni qophaa’ee deebiin itti kenname keessaa Ministar Deetaa Ministeera Nagaa duraanii Obbo Taayyee Danda’aa tokko.
Erga hidhamanii ji’a afur kan tahe Obbo Taayyeen Mudde 1, 201 xalayaa aangoorraa ka’uusaanii agarsiisu waliin yaada Ministira Mummee Abiyiin akkaan qeequ qoodanii turan.
Ministirri Mummee, ''gara jabeessa dhiiga namaan taphataniidha,’’ kan jedhan Deetaan Ministira Nageenyaa duraanii kun, halkanuma barreeffamicha qoodan sana to’annoo jala oolfaman.
Halkan sana hoogganaan (Admin) garee WhatsApp barreeffamoota 23 Obbo Taayyee akkaan qeeqan miseensootaaf ergan. ''Wanbadee’’ Deetaan Ministeera Nagaa kan turan Obbo Taayyeedhaan, jechuun barreefamoota maqaa xureessan yeroo ergan geessituun barreefama Obbo Taayyee Fesbuukii isaaniis jalqabaa waliin qabsiisuudhaan.
Hogganaan garee kanaa, ''hoggansi hunduu, miseensi, deeggaraa fi raayyaan miidiyaa hawaasaa yaada qindaa’e irratti kennaa. Share fi like gochuun garuu gonkumaa dhorkaadha,’’ ajaja jedhu dabarsaniiru.
Maxxansa Obbo Taayyeen qoodan irratti ala 1,00 ol yaadni kennameera. Kunneen keessaa BBC’n 000 adda baasuun yaalii qorannoo taasiseen, barreeffamoonni kara WhatsAppi’chaan ergaman yoo xiqqaate ala 3 maxxansa Fesbuukii Obbo Taayyeen qoodan irratti komanti godhamuu mirkaneesseera.
Qaamoolee 'raayyaa miidiyaa hawaasaa’’ Paartii Badhaadhinaan duulli itti geggeefame keessaa hogganoota TPLF ni argamu
''TPLF Amajji darbe gamaaggama ji’oota fudhate yeroo taasisaa turetti dura taa’aa paartichaa Dr Dabirtsiyoon Gabiramikaa’elii fi miseesa koree hojii raawwachiiftuu kan tahan Fatilwarqi Gabiragizaabiheer waliigalteen Piriitooriyaa akka hin raawwatamneef ''gufuu’’ tahuu isaanii, suuraaleen agarsiisan garee WhatsApp'chaa irratti qoodamaniiru.
Suuraawwan qindaa’uun miseensoota garee sanaatiif qoodaman 20 keessaa, Dr Dabirtsiyooniifi ''fakkaattiiwwansaa’’ jechuun kan eeraniin aanga'oota Naannoo Tigraayi biroos suuraan ''vaayirasii uummataa’’ jechuun waamu keessa aragama. Suuraan kunis ala 330 Fesbuukii irratti maxxanfamuu BBC’n mirkaneesseera.
Dhuma baatii Onkololeessa irra bulchiinsa yeroo Naannoo Tigaaraayiitti, aangoo Ittaanaa pireezidantiin gorsaa pireezidantii kan turan, Alam Gabirawaahiid, barreessaa olaanaa kaabinee kan turan Obbo Amaanu’el Asaffaa fi aanga'oonni kaan itti gaafatamummaa isaanii irraa yeroo ka’anitti, suuraaleen qabiyyee wal-faakkaataa garee WhatsApp'chaarratti ergamanii turan.
Suuraalee Fesbuukuu irratt qoodamuu isaanii BBC’n mirkaneesse 20 keessaa tokko Obbo Gabirawaahidiin ''Ye Tigraayi naqarsaa’’ jechuun waama.
Kanneen faallaa mootummaa dhaabbatan ilaalchisee ibsawwan akkasiitti fayyadamuun akka ''haasaa jibbaatti,’’ lakkaa’amuu danda’u, Qorataan Seera Mirgoota Namoomaa Dijitaalii Dr Yohaannis Inniyew dubbatu
Mootummaan Ministira Mummee Abiy Ahimad, ''haasaa jibbinsaa fi odeeffannoo sobaa ittisuu fi to’chuuf'' jechuun bara 2012'tti labsii baasee ture. Seerri kun kan bahe, ''jibbaa fi haasaa sobaa tahe jedhamee tamsaafaman seeraan ittisuu fi to’achuudhaaf barbaachisaa ta'ee waan argameefiidha,’’ jedha labsii bahe kun.
Haasaan jibbiinsaa miidiyaa hawaasaarrattitamsa'u yakkaan akka nama gaafachiisu kan kaa'e labsiichi, namni kana raawwatee argame hidhaa salphaa hanga waggaa sadiin akka adabamu kaa’eera.
Itti gaafatamummaa seeraa kana kan kaasan Dr Yoohaannis, ''haata'u malee mootummaan yakka kana gaafa raawwatu seerichahojiirra oolchuun baayyee rakkisaadha. Filmaanni jiru waan soroorsuu qabiyyee jedhamudha,’’ jechuun murteessummaa tarkaanfii kaampaanii miidiyaa hawaasummaan fudhatamu ibsu.
Waajjirri Mummeen Paartii Badhaadhinaa deebii BBC'f kenneen, miseensonni paartiichaa ''odeeffannoo sobaa fi haasaa jibbaa ta'utti fudhatamu danda'anii fi suuraalee tamsa’aniiru,’’ jedhamuun ''gonkumaa ibsa fudhatama hin qabnedha,’’ jechuun waameera.
Tamasa’uu oduu sobaa fi haasaa jibbaa, ''akka paartiittis ta'e akka mootummaatti kan cimsinee irratti qabsoofnuufi dhimma guyyaa guyyaadhaan itti saaxilamnu kana sababiin nuti raawwachu itti dandeenyu hin jiru,’’ jechuun sochii akkanaa keessatti hirmaannaa akka hin qabne ibseera.

''Farajjuun kunis fudhatama kan hin qabnee fi kan yaadaafi gochaa paartichaa hin ibsinedha,’’ jechuun deebii isaarratti kan barreesse paartiichi, paarlaamaadhaanis seera ''oduu jibbaafi odeeffannoo sobaa ittisuuf ni dandeessisa jedhu'' bakka raggaasisuus yaadateera.
Waajirri mummee paartichaa, '''oduu jibbaa' fi 'odeeffaanoo sobaa' dhiisiitii, wayita dogoggorriwwan ixxiqqaan uumamuyyuu sirreeffama gochaa kan turre yoo ta'u, ammallee gochaawwan wal-fakkaataan raawwataman qabatamaan yoo argine kanneen raawwatan irratti tarkaanfii sirreeffamaa barbaachisu akka fudhannu ibsuu barbaanna,’’ jedheera.
Dhaabbata haadhoo Feeuukii tahe Meettaan sochiileen miidiyaa hawaasaa bifa duulaatiin gaggeefamu, ''amala sobaa qindaa’e’’ (coordinated inauthentic behaviour) jechuun waamani.
BBCn odeeffannoowwan qorannoo raayyaa miidiyaa paartii badhaadhinaarratti taasiseen argate irratti bu'uurreeffachuun, gaaffii Meetaadhaaf dhiyeesseen, kaampaaniichi amalli sochii qindaa'e jiraachu mirkaneessuun, tarkaanfii fudhachu ibseera.
Meettaan deebii karaa imeelitiin BBC'f kenneen, ''akkaawwuntiiwwan sobaatti’’ fayyadamuudhaan maxxansawwan ''yaadoota gaarii agarsiisan baayyeen, fuula Fesbuukii ''mootummaa Itoophiyaa’’ irratti kennamusaanii qorannoo taasiseen irra qaqabu beeksiiseera.
Gochaan kunis imaammata amala sobaa dhaabbatichaa akka sarbuu kan ibse kaampaanichi, ''tuuta akkaawuntiiwwaniifi fuula Fesbuukii’’ (a cluster of accounts and Pages) haquu isaa deebiicha irratti himeera.
Dubbii himaan Meettaa, ''tuuta akkaawuntiiwwaniifi fuula Fesbuukii imaammata amala sobaa dhaabbatichaa sarban haqneerra. Akkaawwuuntiiwwaniifi toorriwwan muraasnis sirna awutoomaatikii keenyaan haqaman.
Akkaawwuntiiwwan kunneenis maxxansawwan waa’ee mootumamaa Itoophiyaa haala nam-tolcheetiin tamsaasni isaanii akka dabaluuf yaada kennuun, sochii dogongorsaa fi hin barbaachifne (spammy) keessatti hirmaataniiru'' jechuun deebii imeeliin BBC'f kennaniin ibsaniiru.
Dhaabbatichi garuu sarbama jedhe kanaan wal qabatee akkaawuntiiwwan fesbuukii fi fuulawwan meeqa akka haqe hin beeksisne. BBC’n yeroo dhaabbatichaaf gaaffii dhiyeessetti, akkaawuntiiwwan akka fakkeenyaatti eeraman, guyyaa deebiin imeelii itti kennamee fi guyyoota isaan dura turanitti maqfamuusaanii BBC’n hubateera.
Miseensa ''raayyaa miidiyaa hawaasaa’’ tahuun olola Paartii Badhaadhinaa tasaasuuf akkaawuntiiwwan hojiirra oolan qabiyyee wal-fakkaataa qabaachuu BBC’n akkawuntiiwwan 200 irratti hordoffii taasiseen arguu danda’aeera.
Kaayyoo kanaaf akkaawwuntiiwwan faayidaarra oolan kan fayyadamtoota Feesbuukii idilee hin fakkaatu. Suuraafi barreeffamni enyummaa isaanii ibsus irraa hin mul’atu.
Akkaawunttiiwwan kunneen irratti suuraan Ministira Mummee Abiyyiifi Kantiibaa magaalaa Finfinnee suuraawwan piroofaayiliidhaaf faayidaarra oolan keessaati.

Suuraawwan ''meeggaa piroojektii’’ mootummaa ijaarsise akkasumas barreeffamoota Paartii Badhaadhinaafi Raayyaa Ittisaa ibsan suuraa waliin qindaa’an akkaawuntiiwwan kunneen irraatti suuraa piroofaayilii tahuun mul’atu.
Akkaawuntiiwwanmuraasni immoo suuraawwan artistoota dubartootaa akka fayyadamanis BBC’n bira gaheera.
Miseensonni ''raayyaa miidiyaa hawaasaa’’ waajira Paartii Badhaadhinaa Naannoo Oromiyaa xiqqoo hin taane haala adda taheen suuraa wal-fakkaatu akkaawuntii isaaniif fayyadamu.
Akkaawuntiiwwan suuraa wal-fakkaataa fayyadaman caasaa godina tokko jalatti miseensota argaman tahuu horodoffii taasiseen BBC’n hubachuu danda’eera.
Caasaawwan mootummaa naannichaa keessaa godinooti Arsii Lixaafi Baalee Baha ani argamu. Miseensonni raayyaa miidiyaa hawaasaa godina Jimmaa immoo suuraa ''raayyaa miidiyaa hawaasaa dijitaalaa’’ barreeffama jedhu kan Afaan Ingiliiziifi Afaan Oromoon irratti barreeffame fayyadamu.
Meettaan mootummaa Itoophiyaa wal-qabatee akkaawuntiiwwan akkasii irratti tarakaanfii fudhachuun kana jalqabaa miti.
Yeroo waraanni Tigraayi jalqabe akkaawuntiiwwan Feesbuukii 2, fuula(pages) 9 fi gareewwan 2 imaammata ''amala sobaa qindaa’e’’ cabsan akka haqe ibsee ture.
Akkaawunttiiwwan kunneen ergaawwan qoodan keessaa, ''Paartii Badhaadhinaa fi Ministira Mummee Abiy Ahimadiin,’’ kan ilaallatan akka tahe Meettaan Waxabajjii 2013 A.L.I ibsa baaseen beeksisee ture.
Meettaan qorannoo taasiseen gochaa kana duuba namoonni dhuunfaa turan, Bulchiinsa Netwarkii Odeeffanoofi Nageenyaa waliin hidhata qabaachuus akka hubate ibseera.
Mootummooti ''amala sobaa qindaa’e’’ fayyadamuun dhimma haaraa akka hin taane qorataan mirga namoomaa dijitaalaa Dr Yohaannis ni dubbatu. Keesumaa yeroo rakkoon siyaasaafi nageenyaa isaan mudatu mootummaan hawaasa to’achuuf sochii akkanaa akka raawwatan ibsu.
Qorattoonni Okisfard bara 2012 olola gama qindaa’een miidiyaa hawaasaa irratti tamsa’anirratti qorannoo taasisaniin yoo xiqqaate biyyoota 70 keessatti duulli siyaasaa dogoggorsaa akka adeemsifamu mirkaneesseera. Qorannicharratti biyyoota eeraman keessaa Itoophiyaan tokko.
Misitirrii Mumee Abiy gara aangotti dhufuu isaaniin dura yeroo jiru akkaawuntii sobaa fayyadamuun duula miidiyaa hawaasaa irratti adeemu kan qindeessu dhaaba mootummaa akka tahe qorannoo kanarratti gabaafameera.

Akkaawuntiiwwan kunneenis mootummaaf deeggarsa kennuufi mormitoota qeequuf tajaajilaaf oolu jedha.
Miseensota ''raayyaa miidiyaa'' durgoon kaffalamuuf
Koominikeeshinii mootummaa keessatti waggoota 1f kan hojjatan maddi BBC’tti haasa’an tokko, miseensota paartii fayyadamuun olola mootummaa Feesbuukii irratti maxxansuun bara dhumaa ADWUI karooraa dhimma ture akka tahe dubbatan.
Mootummaa irratti mormiin adeemsifamu ALI bara 2008 jabaachaa gaafa dhufu Waajirri Dhimmoota Koominikeeshinii Mootummaa duraanii ogeessota dhimmi kun ilaallatuuf leenjii kennaa akka ture yaadateera.
Tahus paartilee ijoo afur ADWUI hudeessan gidduutti garaagartummaan yaadaa bal’achaa waan dhufeef karoora ture hojiirra oolchuun hamma barbaadame hojiirra akka hin oolle himan.
Duulli yeroo amma hojiirra oolaa jiru kuni isa wagga saddeet dura karoorfame waliin akka wal-fakkaatu kan ibsan maddi kuni.
Mootummaafi Paartiin Badhaadhinaa karoora qophaa’e ture hojiirra oolchuun ''milkaa’aniiru’’ jedhanii akka yaadan eeraniiru.
Duula bifa qindaa’een geggeefamuun dabalataan, itti fayyadama miidiyaa hawaasaa fi sirna shallaga(algorithm) irrattis leenjiin ammallee Paartii Badhaadhinaa caasaa garagaraa keessattii kennamu.
Miseensota paartii fi ogeessota koominikeeshiniif leenjiiwwan kennaman keessa xiyyeeffannoon tokko, ''piroojektiiwwan misoomaa biyyattii’’ ilaallatan irratti ''uummataaf odeeffannoo qaqqabsiisuu,’’ kan jedhu dha.
Waajirri mummee Paartii Badhaadhinaa deebii BBCf kenneen, ''maalummaaafi fayyadama miidiyaa hawaasaa, akkatti appiiwwan hojjetan isaaniin fayyadamuun ammo uummataaf akkamiin odeeffannoo tamsaasuun akkan danda’amu,’’ hogganootaafi miseensotaaf leenjii akka kennu ibseera.
Paartiin kuni, ''kuni ammoo addunyaarratti hawaasni siyaasaa hundi waan raawwatuufi raawwachuu qabu tahuu amana,’’ jechuun hojimaata baratame tahuu ibseera.
Fuula Feesbuukii mootummaafi paartii irratti hordoftootaafi hirmaattota baayyisuuf tattaaffii taasifamu waajiraaleen Finfinnee keessatti argaman lama namoota, ‘like’, ‘share’ fi ‘comment’ hamma qacaruutti akka gahame maddeen waajiraalee lamaan kanaa BBCtti himaniiru.
Haala kanaan fuula Feesbuukii isaanii iarratti hordoftoota baayyifachuuf Waajira Dargaggootaafi Ispoortii Magaalaa Finfinnee, akkasumas Waajira Misooma Humna Namaa fa’i dha.
Maxxansawwan BBC’n fuula Feesbuukii kunneen irratti hordoffiifi xiinxala taasiseen maddeen irraa odeeffannoo argame wajjin wal-fakkaataa dha.

Waajirri Ispoortiifi Dargaggoo magaalaa Finfinnee dhimma kanarratti gaafatame, ''namoonni dhuunfaa dhimma kanaaf qacaramaniifi kaffaltii argatan hin jiran,’’ jechuun wakkaateera.
Waajirichi itti fayyadama miidiyaa hawaasaa ilaalchisee dargaggootaaf leenii akka kennu daarektarri quunnamtii hawaasaa waajirichaa Obbo Mahaarii Tamasgan leenjii kana warreen fudhatan akkaawuntii isaanii irratti hirmaannaa akka qaban ibsaniiru.
Obbo Mahaariin, ''dargaggoonni waajira keenyaa waliin hojjetaniifi maatiin ispoortii fuula Feesbuukii dhaabbatichaa irratti qabiyyeewwan bahan ‘like’ fi ‘comment’ ofiisaanii akka kennan leenjii kennina.
Kanneen leenjii fudhatan keessaa irra deddeebiin ‘comment’ gochaa jiraachuu malu,’’ jedhaniiru.
Itti gaafatamaan Waajira Pabliik Sarviisiifi Misooma Humna Namaa Dr Xaasaw Gabireen immoo gochaan akkasii jiraachuusaaf, ‘’odeeffannoo hin qabu,’’ jechuun dubbataniiru.
Dhimmicha qulqulleessuun deebii akka kennan waadaa seenanillee deebisanii bilbilaan argamuu hin dandeenye.
Akka BBC’n maddeensaarraa odeeffatetti namoonni 11 gochaa kana raawwachuuf qacaraman fuula Feesbuukii waajiraalee lamaanii irratti hirmaannaa taasisu.
Namoonni kunneen miseensota paartii akkasumas aanaalee fi kutaalee magaalaa keessatti ogeessa koominikeeshinii tahuun hojii idlee kan raawwatan tahuu maddi dhimiichaaf dhihoo tahan BBCtti himaniiru.
Maddi kun dabaluunis, ''namni tokko akkaawuntii Feesbuukii saddeetii hanga kudhanii qaba'' jedhan.
Qabiyyeewwan waajiraaleen kunneen maxxansan fuudhaniis fuula kurnan sanarrattis ‘share’ godhu. Namoonni kunneenis gochaa isaanii kanaaf ''bifa durgoodhaan’’ akka kanfalamu maddi kuni ibsaniiru.
BBC’n qabiyyeewwan Feebuukii horofuuf kan sirni ''Crowd Tangle’’ fayyadamuun odeeffannoo argateen baatii Sadaasaa fi Mudde irraa jalqabee maxxansawwan Feesbuukii waajiraalee lamaan irratti lakkoofsi akkaawuntiiwwan hirmaatanii haala olaanaa taheen dabaleera.
Yeroon kun maddeen keenyi namoota dhuunfaa kunneeniif qacarriin rawwatame yeroo jedhan waliin wal-fakkaata.
Yeroo baayyee qabiyyeewwan fuula sana irratti bahan irraa akkaawuntiiwwan komeentii kennan wal-fakkaatoo yoo tahan, fuula waajiraalee lameen kanarrattis akka hirmaatan mirkaneesseera.
BBC’n gochaa kanarratti kan hirmaataniifi akkaawuntiiwwan 1 namoota dhuunfaa lama wajjin hidhata qaban adda baaseera.

Qorataan Mirga Namoomaa Dijitaalaa Dr Yohaannis, qabiyyeewwan mootummaa wajjin hidhata qaban irratti haala qindaa’een komeentii kennuun, ''yaada bilisaan ibsachuu osoo hin taane kaanin qaanessuu dha.
''Sochiin akkasii, fayyadamtoota ukkaamsuu, kanneen si’oominaan barreessan waltajjii kanarraa akka bahan gochuu akkasumas yaada ofii bilisaan ibsachuufi miga odeeffannoo argachuu akka daangessu,’’ eeraniiru.
Qondaalli siyaasaa ABO Obbo Battee Urgeessaa torban darbe Ebla 9, 202 iddoo dhalootasaa magaalaa Maqii keessatti rasaasaa hedduun ajjeefamee argame.
Ajjeechaan nama siyaasaa kan qabsoo karaa nagaatiin godhamurratti ejjennoo jabaa qabu kanarratti raawwatame dhaabasaa dheekkamsiiseera. Dhaabbileen mirgaaf falmaniifi hawaasni idil-addunyaas qorannoon walabaafi amanamaan akka gaggeefamu dhaamaniiru.
Misieensonni maatiifi dhaabnisaa ajjeechaan kaaf qaamolee nageenyaa mootummaatti quba yoo qaban, mootummaan ammoo ajjeechaa kanaaf gaafatamaa godhamuunsaa fudhatama akka hin qabne ibsuun akeekkachiiseera.
Qajeelchi Poolsiii magaalaa Maqii ajjeechaa Obbo Batteen walqabatee namoota shakkamoodha jedhe 13 to'annoo jala oolchuu dhuma torbee darbee beeksisee ture. Wal dhibdeen maatii sababa ajjeechaa kanaa akka ta'uu malus Kintiibaan magaalitti himanii turan.
Hooggantoonni dhaabbilee siyaasaa mormituu BBCn dubbise garuu ajjeechaan Obbo Battee ''ajjeechaa siyaasaati'' jedhu.
Dura taa'aan dhaaba Kongiresii Federaalawaa Oromoo Piroofeesar Mararaa Guddinaa, Dura taa'aan dhaaba Hibiri Itoopihyaa Obbo Girmaa Baqqalaa fi walitti qabduun tokkummaa dhababilee siyaasaa Itoophiyaa duraanii Dr. Raaheel Baafee BBCtti dubbataniiru.
Piroofeesar Mararaan ''ajjeechaan kun siyaasuma. Maaliifii namni magaala seenee hooteelaa baasee halkan sanaan magaala keessatti ajjeesee, kana gochuu kan danda'u qaama humna qabuudha,'' jedhan.
Bakka eegdonni hoteelaa fi poolisiin magaalaa jirutti dhibdee dhuunfaatiin namni ajjeechaa kana raawwachuu danda'a jedhanii akka hin amanne dubbatan.
''Qaama humna qabuudha. Isa asiifi achi irra diban osoo hin taane, irra deddeebinee qaama walaba ta'een haa qulqullaa'u kan jedhamaa jirus kanumaafidha.''
''Lola dhuunfaatii jedhamee inni odeeffamu homaa waan dhugaatti siiqu miti. Qaama humna qabuun raawwatamuunisaa wal hin gaafachiisu'' jedhu Piroofeesar Mararaan.
Walitti qabduun tokkummaa Dhabbileee Siyaasaa Itoophiyaa duraanii Dr. Raaheel Baafee gama isaaniin, ''Anaaf ajjeechaan kun kan siyaasaati. Namoota akka Battee irratti ajjeechaan raawwatu kanuma siyaasaati,'' jedhu.

''Siyaasni Itoophiyaan gaggeessitu tokko aarsaa godhanii tokko oolchuudha. Kanatu rakkoo ta'e. Rakkoo isa tokkoof kan biraa ajjeessuun furmaata gochuutu barame,'' jedhu.
Dura taa'aa dhaaba Dimookraatawaa Hibir Itoophiyaa kan ta'an Obbo Girma Baqqalaas akkasuma ajjeechaan Obbo Battee ''kan siyaasaatiin ala ta'uuf carraan qabu baayyee dhiphaa ykn gonkumaa kan hin taanedha,'' jedhan.
''Tilmaamaan olitti ajjeechaan kun kan siyaasaa malee waan biraa hin ta'u jedheen yaada,'' jedhan.
Obbo Battee wajjiin waggoota muraasa darban waliin walitti dhiyaatanii hojjachuu kan himan namni siyaasaa buleessi kun, sababoota ajjeechaan kun kan siyaasaa malee waan biraa miti jedhaniif tarreessaniiru.
''Tokkoffaa nama siyaasaa beekamtii guddoo qabu, qabsoo karaa nagaatti amanu dhiisiitii namni kaanuu ajjeefamee magaalaa baafamee yoo gatamu mootummaan eessa jira? Halkan sanaan reeffa fuudhanii magaalaa baasuuf akkamiin onnee argatan? Rasaasni gara kudha wayii Battee rukute yeroo dhukaafamu qaamni poolisii ykn nageenyaa naannichaa eessa ture?'' gaaffilee jedhan kaasisa.
Sababa kanaatti ani ajjeechaa siyaasa malee waan biraa ta'a carraan jedhu baayyee xiqqaa ykn ta'uu kan hin dandeenye jedheen amana.''
Ajjeechaan Obbo Battee qabsoo karaa nagaa godhamurratti waan akeeku qabaa?
Obbo Girmaa Baqqalaa kaayyoon ajjeeechaa kanaa ''ejjennoo siyaasaa itti amananiif cimanii dhaabachuun akka balaa qabu agarsiisuudha'' jedhu.
''Ejjennoo siyaasaa ifa ta'e qabachuufi ejjennoo kanaaf cimanii dhaabachuu laaffisuudhaaf tarkaanfii addabaabayiitti fudhatameedha,'' jedhanii ibsu.
''Warra kaan sodaachisuuf wanta godhamedha. Namoonni siyaasaa akkasii ejjennoo qaban hammuma baayyatan dhiibbaa mootummaarraan gahuu danda'an xiqqeessuuf waan godhameedha jedheen yaada,'' jechuunis tilmaama isaanii ibsan.
Ji'oota dura gabaasa dhima hiree qabsoo nagaa Itoophiyaa keessa jiruu irratti Obbo Battee dabalatee namoota siyaasaa adda addaa dubbifne, haalli jiru yeroo kamuu caala hammaachaa dhufuu ibsanii ture.
Obbo Batteen wayita san yaada kennaniin dirreen qabsoo karaa nagaa hammamuu dhiphachaa deemu ''abdii kutuudhaaf hin jirru'' jedhanii ture.
Amma ''ajjeechaan siyaasaa' nama yaadaan muggutuun dhageettii horataa jiru kanaa qabsoo karaa nagaaf abdii kutachiisaadha'' jedhu namoonni dhaabbilee siyaasaa sadan dubbifne.
Piroofeesar Maraaraa, Obbo Girmaa fi Dr Raaheel ajjeechaan kun qabsoo karaa nagaan godhamu abdii kutachiisuutti adeemaa jiraa jechuun ibsan.

Obbo Girmaan, ''ajjeechaan kun dirreen siyaasaa hangam takka akka dhiphate ykn cufameera sadarkaa jedhurra gahee namoonni dhaaba siyaasaa keessatti hooggansa olaanaarra jiraniiyyuu lubbuu isaaniif sadarkaa wabii hin qabnerra gahuu agarsiisa,'' jdhan.
Dabaluunis, ''hooggansi siyaasaa kaanis kaka'umsi qabsoof qaban laafee qabsoorraa booda akka jedhan dirqisiisa'' jedhu.
Piroofeesar Maraaran seenaan Itoophiyaa barkurnee hedduuf ture sagalee mormii cufuu fi ukkaamsuun as gahee jechuun, ta'us ''qabsoon takkaa hin dhaabbanne'' jedhu.
Uummanni miiliyoonaan dararaamaa jira, namoota siyaasaa ajjeesuu fi hidhuun, dararuun miidhaa qabaatus, qabsoo karaa kamiinuu godhamu kan jabeessu malee kan laaffisu miti,'' jedhu.
''Rakkoo siyaasaa jiru walitti dubbatanii furuuf abdii kutachiisa. Rakkoo siyaasaa boruufi iftaaniif walitti dubbachuu mataasaa rakkisaa taasisa. Mariif karaa cufaa deema malee karaa saaqaa hin deemu.''
Seenaa siyaasa Itoophiyaa darbee yaadachuun, bu'aan haala akkasiin dhufu eenyuunuu hin fayyadne hin fayyadus'' jedhan Piroofeesar Mararaan.
Dr Raaheel Baafee gama isaaniin, ''yoomiyyuu caala abdii kutachiisaa dhufus furmaanni waaraana karaa kana qofa waan dhufuuf itti cichuu malee falli biraan hin jiruu jedhu.
Aangoon mootummaa filannoofi muggutii yaadaan malee waraanaan ta'uun irra hin jiruu kan jedhan Dr Raahel, qabsoo karaa nagaa gara filannoo walabaatti geessuun akka dhufu himan.
''Sirnichi nageenya biyyaafi aangoo uummataaf deebisuuf dursa hin kenniine. Santu yaadessaadha. Batteen bosona galee meeshaa waraanaa waliin argamee miti kan ajjeefame. Sababa yaada isaatiif ajjeefame. Kanaaf qabsoo karaa nagaa kun abdii kutachiisaa jira. Garuu ammoo itti dhiisuu hin dandeenyu,'' jedhan.
Paarlaamaan Itoophiyaa waa’ee kiraa mana jireenyaa ilaalchisee seera haaraa baaseera. Labsiin haaraa ragga’e kuni namoota mana jireenyaa kireessan irratti dirqama haaraa kaa’e.

Yoo kuni irra gahame qajeelfama qaama to’atuun adabbii darbuun ala namni haaraa kireeffatu gatiin gara duraanitti akka deebi’u iyyatee deebisiisuu danda’a jedha.

Kanaan ala namni kireessu gatii mana kiraa kan ji’a lamaa ol dursee kireeffataan akka kanfalu gaafachuu akka hin dandeenye seerichi ni ibsa.
Akka kana dura namoonni afaaniin qofa waliigalanii mana hin kireessan, yookiin hin kireeffatan.
Namni kireessuufi kireeffatu sanadoota kaffaltii mana kiraa qindeessanii kaa’uuf dirqama qabu jedha.
Waliigaltee mana kiraa kamuu barreeffamaan taasifama. Kunis, qaama to’atuun galmaa’uufi mirkanaa’uu akka qabu dirqamsiisa.
Akkasumas, qaamoleen lamaan waliigalteen erga raawwatee guyyoota 30 keessatti ‘’waliigalee dhiheessanii akka mirkanaa’uufi galmaa’u gochuuf dirqama qabu,’’ jedha.
‘’Waliigalteen mana kiraa hin mirkanoofneefi hin galmoofne akka waliigalteen hin raawwanneetti ilaalama,’’ jechuunis ifa godha.
Osoo mana gadhiisuu feetanoo?
Waliigalteen mana kiraa kan diigamu tokko osoo yeroon waliigaltee hin xumuramiin qaamoleen waliigalan waliigaltee diiguuf yoo waliigalanidha.
Yookiin immoo kireeffataan yoo gadhiisuu fedhe baatii lama dursee kireessaaf beeksisuu qaba.
Gama kaaniin waliigalteen mana kiraa akeekkachiisa malee karaawwan ittin diigamuu danda’us jira.
Yoo nama kireessuufi kireeffatu gidduu rakkoon jiraate qaamni to’atu beeksifamuu qaba.
Qaamni to’atu erga murtee irratti kennee booda komiin yoo jiraate koree hundaa’uun akka ilaalamu ta’a.
Murtee koreerratti qaamni komii qabu dhimmicha mana murtii geessuufi guyyoota 30 keessa murtee akka argatu seerichi ibsa.

Abboomaa Mitikkuu barreessaa olaanaa Gamtaa Barattoota yunivasriitii ta’ee osoo jiruu bara 197tti warraaqsi Dargii aangotti fide dhohuun deemsa qabsoo isaa gara sadarkaa biraatti akka cehu taasise. Yeroon kun yeroo itti ABOn dhoksaan wixinee hundeeffama dhaabaa itti qopheessuu fa’aa eegaleedha.

Baaroon koodii M- jedhuun Abboomaafi namoota shan waliin icciitin Somaaliyaatti erge
Abboomaa duula hawaasa barsiisufi dammaqsuu Dargiin labserraa waamamee akkuma gara Finfinnee dhufeen Barroo Tumsaa fa’aa hojii deeggarsa ABO akka hojjataniif karaa imbaasii Somaalee dhoksaan haala mijeessuun gara Somaaliyaatti isaan erge.
Baaroon yeroo sana dhaabbata Ye Hizbi Dirijjit (Ijaarsa Uummataa) jedhutti qindeessaafi namoota akka Sannaayit Likkeefi Hayilee Fidaa wajjin yeroo gabaabaf hojjatus, yeroo hedduu qabsoo Oromoo qindeessuufi daandii qabsiisurratti akka dabarseyu himama.
Abboomaa Mitikkuu Finfinneetti erga deebi’ee booda “Gadaafi Muhee Abdoo fa’aa wajjin wal-argee, Baaroon haala tokko tokko qiqindeessee jara kana Moqadishootti erguuf karaa imbasii Somaaliyaa haala mijeesse," jechuun namoota jaha wajjin Abboomaan gara Somaaliyaa akka deeme himu.
Yeroo sana Jaarraa [Abbaa Gadaa] fa'aa Moqaadishootti waan hidhamaniif isaan hiiksisuuf garasitti ergaman. "Namoonni jahaan ergaman: Abboomaa Mitikkkuu, Muhee Abdoo, Damisee Tacaanee, Bariisoo (Ibraahim Kaliil)fi Zakkaariyaas nama jedhamuufi namni biroo ammoo Usmaan jedhama.
"Koodii M- jedhamu tolchanii [Baaroo Tumsaa fa'aa] Moqadishootti ergan. Moqaadishoo gaafa gahan Abboomaa Mitikkuu hidhaa seene," jechuun dhimmi deemaniif milkaa'uu didee gara Yaman deemanii achiis simatamuu dhabanii akka deebi'an himu Obbo Dheeressaan.
Yamanirraa gara Moqadishootti deebi'anii achiimmoo tattaaffii Baaroo Tumsaa fa'iin Abboomaa Mitikkuu, Muhee Adoofi Damisee Tacaaneen bara 197 gara dhumaa biyyaatti deebi'uusaanii himu.
Erga Somaaliyaatii deebi'ee booda kora ABO jalqabaa Hagayya 197 taa'ame keessatti Abboomaa Mitikkuu koree giddu-galeessaa ta'ee filayame. Erga koree giddu-galeessaa ABO keessatti filatee hojii dhaaba gurmeessuufi ijaaruu keessatti qoodaa guddaa gumaachaa ture.
Erga Somaaliyatti rakkiin isa mudatee deebi'ee Finfinneetti bara 1977 keessa taateen inni hin eegne isa mudate. Halkan tokko Abboomaan konkolaataa oofee Kazaanchisii ka'ee gara mana abbaasaa Kiiloo jaha Imabsii Masirii cinaatti argamuutti osoo galaa jiruu qondaalli nageenyaa Dargii tokko isa ajjeesuuf isa hordofe.
"Osoo inni konkolaataa oofee galgala gara manaa galuu namni Dargii ajajaa shantamaa ofiin jedhu tokko mooraa Dargii keessaa itti bahee hordofee ajjeesuu barbaade.
Osuma dheesuu mana warraa gahee biraa hafuu didee konkolaataa keessaa bu'ee harka kennadhu gaafa jedhuun Abbooman shugguxii baasee bitaadhan dhukaasee ajjeese," jechuun waajjira poolisii dhaqee harka akka kennate yaadatu.
Abboomaan harka erga kennatee booda qodaalota Dargii keessaa namoonni dhoksaan qabsoo ABO deeggaraniifi Baaroo Tumsaa fa'aa akka inni wabiin bahee ala taa'ee dhimmasaa hordofuuf gargaaruu himu.
Qondaalli Dargii sun Abboomaa Mitikkuu ajjeesuu kan barbaade inni hojii ABO akka gaggeessu irratti bareetii jennee gaafannaan ABOtti shakkee osoo osoo hin taane miseensa Ihaappaa itti fakkaatee ajjeesuuf akka ture himan Obbo Dheeressaabn.
"Warri Dargiis namichatu ajjeesuuf hordofe jedhanii ji'a shanii, jaha hidhamee wabiin bahe. Aniifi Daawud Ibsaa yeroo sana yunivarsiitirraa addaan kunnee hojii hojjanna jennee Naqamtee jirra [hojii ABO dhoksaan hojjatamu].
"Abbaan Caalaa nu dursee Naqamte jira. Sadii taanee achi osoo jirruu Abboomaan dhufee nutti dabalame," jedhu.
Abbaa Caalaa Lataa kanaan dura turtii BBC waliin taasisaniin hojii ABO dhoksaan hojjachuuf maqaa ogeessa diingadee olaanaa jedhuun bulchaa Kutaa Wallagaa yeroo sanaa kan ta'an Obbo Abbiyyuu Galataa haala mijeessanii garas imaluu himan.

Abbaa Caalaa Lataa hojii kanaaf gara Wallaggaa deemanii osoo jiranii Abboomaa Mitikkuu, Obbo Daawud Ibsaafi Obbo Dheeressaa Qixxee fa'aa itti dabalaman.

Mootummaan Naannoo Amaaraa TPLF waliigaltee Piritooriyaa guutummaatti cabsuudhaan marsaa afraffaatiif ummatarratti weerara bane jechuun himate.
Mootummaan Naannoo Amaaraa ibsa har’a Ebla 17, 202 baaseen, “dogongora seenaa kaleessaarraa barachuu kan hin dandeenye TPLF, waliigaltee Piritooriyaa guutummaan guutuutti cabsuudhaan yeroo afraffaatiif ummata keenya weerara,” jedhe.
Naannichi kana kan jedhe dhuma torban darberraa kaasee bakkawwan falmisiisoo ta’an naannoo Raayyaatti walitti dhukaasa guyyootaaf tureen booda magaalotaa humni raayyaafi poolisii Federaalaa, naannawaa magaalaa ammoo hidhattoonni TPLF galuun isaanii erga ibsameen boodadha.
Mootummaan naannicha aibsa baaseen, TPLF kan weerare Raayyaa Allaamaaxaa, Raayyaa Baallaa, Ofilaa, Koram, akkasumas Zaataa ta’uusaanii ibsuun ummanni nananichaa, mootummaan Federaalaafi hawaasni idil-addunyaa gocha kana akka dhaabsisaniif waamicha dhiyeesseera.
Itti-aanaan bulchaa Bulchiinsa Yeroo Naannoo Tigraay Jeneral Taaddasaa Warradaa ibsa kaleessa kennene, “aanaalee Kibba Tigraay fi Tselemt” keessatti bulchiinsawwan Naannoo Amaaraa diriirfamanii turan diiguudhaaf mootummaa Federaalaafi bulchiinsa yeroo naannichaa gidduu waliigalteerra gahamuu dubbatee ture.
Gaaffiin eenyummaa kan irratti ka’aa tureefi osoo waraanni Kaaba Itoophiyaa hin jalqabiin dura bakkawwan naannoo Tigraay jala turan, waraanaan booda to’annoo humnoota Amaaraa jala galuun isaanii ni yaadatama.
Kanaan dura mootummaan Federaalaa lammileen bakkawwan falmisiisoo ta’anii buqqifaman akka deebiyaniifi buchiinsa ofiisaanii akka diriirfataniifi yeroo itti aanutti ammoo murtee ummataa gaggeessuudhaan gaaffiin ‘naaf malaa’ deebii akka argatu ibsee ture.
Jeneraal Taadddasaa Warradaa ibsa kenne keessatti, gareewwan hidhatanii bakkawwan kanneen keessa socho’an hidhannoo akka hiikkatan akka taasifamu eeruun, murtee mootummaan Federaalaa bakkawwan falmisiisoo ta’anitti rifarandamiitu gaggeeffama jedhe ejjennoo bulchiinsa yeroo Tigraay akka hin taane dubbateera.
Weerarri anarratti gaggeeffameera kan jedhe mootummaan naannoo Amaaraa ibsa isaan, bakkawwan falmisiisoo ta’aan ilalachisee karaa mootummaa Federaalaa bulchiinsa yeroo Tigraay waliin marii taasisuuf yaalii baayyeen taasifamanillee, “TPLF garuu filannoo nagaa kana diduun bakkawwan gaaffiin eenyummaa irratti ka’an akka Raayyaa Allaamaaxaa, Raayyaa Baallaa, Koramiifi Zaataa” weeraree qabachuusaa ibseera.

''Kuni hin ta'u taanaan mootummaan naannoo Amaaraafi ummatni, obbolaasaa lammiilee Itoophiyaa biroo waliin ta'uun biyya diigamuurraa baraaruufi ummata keenyaas ittisuuf akka dirqamnu beeksisuu barbaanna'' jechuunis akeekkachiiseera.

Haadha qabeenyaa Kabaanaa dizaayiniifi dubartoota kampaanicha keessa hojjetanMADDA SUURAA,KABAANAA
Kaabaanaan, dizaayiniifi hojii leezerii yeroo boqonnaa ofiitti jaalala waan sana hojjechuuf ‘hobby’ qabanirraa kan ka’een hojjechuun dhalatee namoota baayyeef carraa hojiifi abdii uumedha.
Samahaal Gu’uush Kaabaanaan hojii yeroo boqonnaashee ture. Amma hojjettoota 100 ol kan qabu yoo ta’u, bu’aawwan gogaafi kallee barbaachisoo omishuun gabaa biyya alaatiif erguun alatti, dubartoota ogeeyyii ta’an xiqqaa hin taaneef carraa kan uumeedha.
Finfinneetti kan guddatte Samahaal osoo waggaa torba dura dhaabbata Kaabaanaa jedhu bantee hojii leezerii hin jalqabiin dura, wayita Yunivarsiitii galtee Arkitekcharii barataa turtetti gogaa irraa faayaawwan harkaa hojjechaa turte.
Erga Yunivarsiitiirraa eebbifamteen booda dhaabbata Arkitekchariitti qacaramtee osoo jirtuu, cinaatti hiriyyootashee biroo waliin ta’uun biizinasiiwwan sadii jalqabdee akka barbaaddetti hin milkaa’iin hafeera. Haata’u malee, carraaqqii taasiftu addaan hin kutne.
Samahaal yeroo boqonnaa qabdu hojii borsaa gogaarraa hojjetamaniif oolchuudhaan dhaabbata Kaabaanaa leezer yeroo jalqabdetti, jireenya dubartii jalqabaa qacarte fooyya’ee arguun hojiishee yeroo guutuu irratti bobbaatu taasifachuuf murteessite.
